Генератор магазинов 2

Доступные функции и фильтры

shuffle

список|shuffle

{{ ["раз","два","три"]|shuffle|join("-") }}

yasyn

строка|yasyn(имя словаря, расстояние между словами, генерировать шаблон)

{{ "фраза"|yasyn("internal") }} - для встроенного словаря
{{ data.2|yasyn("stroyka.yas", 1) }}
{{ "фраза"|yasyn("stroyka.yas", 2, True) }}

by_dict

строка|by_dict(имя json словаря)

{{ "3"|by_dict("fingers.json") }}
{{ "5x5"|by_dict("sauna.json") }}

Для конвертирования словарей версии 1 см. \build\exe.win32-2.6\convert.cmd

choice

choice(список)

{{ choice(["раз","два","три"]) }}

randint

randint(min,max)

{{ randint(1,100) }}

in_words

значение|in_words

{{ "55.5"|in_words }}

nl2br

строка|nl2br

{{ "генератор\nмагазинов"|nl2br }}
